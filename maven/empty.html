JDOM Placeholder Javadoc

This is a placeholder docment to satisfy the OSS requirements of Maven-Central: OSS Projects must include Javadoc and Source artifacts.

The reality is that the JavaDoc for the JUnit test cases would be odd to say the least, and the JDOM Contrib repositories are woefully out of date and inadequate.

In each case reading the source code is a much better idea, and is what you would need to do anyway. Even though this JavaDoc artifact is just a placeholder, the sources artifact is complete, and should be used as a (better) alternative.

The core JDOM jar though has comprehensive JavaDocs, and both the Source and JavaDoc artifacts are fully populated (unlike this...).
